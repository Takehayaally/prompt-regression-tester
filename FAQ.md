# FAQ

## Does this call an LLM API?

No. This MVP validates saved outputs locally. It does not send prompts or outputs to any external service.

## What is the paid bundle for?

The free version proves the workflow. The paid bundle is for teams that want ready-made examples, report templates, and commercial-use packaging.

## Can it test live model responses?

Not in this MVP. Live-provider adapters may be added later if there is enough demand.

## Refund boundary

Refunds are reasonable when the included examples do not run as described. Custom integrations, private prompt review, or provider-specific debugging are not included.
